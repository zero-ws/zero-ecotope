### 2026-03-03 00:23 - instance.yml 支持 init 配置覆盖菜单属性

#### 需求描述

在 `instance.yml` 中追加 `init` 配置节点，用于覆盖 `XMenu` 的属性。使用 `JsonObject.mergeIn()` 方法实现属性合并。

**配置示例：**
```yaml
running:
  78fce5a2-17f3-4dac-a75c-7e751595015c: app.micro.hotel

init:
  menu:
    name: zero.cm
    uri: /ambient/customer/corporation
    text: 客户管理
  app:
    # 应用级别的配置覆盖
```

#### 解决方案

1. **InstApps 接口**：新增 `ioInit()` 方法，用于读取 `instance.yml` 的 `init` 节点
2. **InstAppsLoad 实现**：实现 `ioInit()` 方法，从 classpath 或文件系统加载 `init` 配置
3. **BuildApp**：新增 `loadInitConfig()` 方法，加载 `init` 配置并传递给 `BuildMenuLoader`
4. **BuildMenuLoader**：
   - 新增 `initConfig` 字段，存储 `init` 配置
   - 修改构造函数，接收 `initConfig` 参数
   - 新增 `applyInitMenuConfig()` 方法，使用 `mergeIn()` 覆盖菜单属性
   - 在 `loadMenuFromFile()` 中调用 `applyInitMenuConfig()`

#### 代码改动

**文件：** `InstApps.java`
- 新增 `ioInit()` 方法声明

**文件：** `InstAppsLoad.java`
- 实现 `ioInit()` 方法，读取 `instance.yml` 的 `init` 节点

**文件：** `BuildApp.java`
- 新增 `loadInitConfig()` 方法
- 修改 `run()` 方法，加载 `initConfig` 并传递给 `BuildMenuLoader`

**文件：** `BuildMenuLoader.java`
- 新增 `initConfig` 字段
- 修改构造函数和 `create()` 方法，接收 `initConfig` 参数
- 新增 `applyInitMenuConfig()` 方法：
  - 检查 `init.menu` 配置是否存在
  - 通过 `name` 字段匹配当前菜单
  - 使用 `Ut.serializeJson()` 将菜单对象转为 `JsonObject`
  - 使用 `mergeIn()` 方法合并配置
  - 使用 `Ut.deserialize()` 反序列化回 `XMenu` 对象
  - 复制所有字段到原菜单对象
- 修改 `loadMenuFromFile()` 方法，在填充全局字段后调用 `applyInitMenuConfig()`

#### 验证结果

- ✅ 编译成功：`mvn clean compile -DskipTests`
- ✅ 支持从 `instance.yml` 的 `init.menu` 节点覆盖菜单属性
- ✅ 使用 `JsonObject.mergeIn()` 方法实现属性合并

### 2026-03-02 23:03 - 菜单文本后缀处理

#### 需求描述

菜单文件名中如果包含 `_W` 或 `_P` 后缀，text 属性不应包含这些后缀。

**示例：**
- 文件名：`1000_用户管理_P.yml` → text 应为 `用户管理`（去除 `_P`）
- 文件名：`2000_系统设置_W.yml` → text 应为 `系统设置`（去除 `_W`）

#### 解决方案

修改 `parseFileName()` 方法，在解析文件名后去除 text 中的 `_W` 和 `_P` 后缀。

#### 代码改动

**文件：** `BuildMenuLoader.java`

修改 `parseFileName()` 方法：
```java
// 去除 text 中的 _W 和 _P 后缀
if (text.endsWith("_W")) {
    text = text.substring(0, text.length() - 2);
} else if (text.endsWith("_P")) {
    text = text.substring(0, text.length() - 2);
}
```

#### 验证结果

- ✅ 编译成功：`mvn clean compile -DskipTests`
- ✅ 菜单 text 属性正确去除 `_W` 和 `_P` 后缀
